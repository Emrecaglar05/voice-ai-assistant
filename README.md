## 🎯 About

> Voice AI projects including a **RAG-powered voice assistant** and a **modular Text-to-Speech training pipeline**. Combines speech recognition, LLM reasoning, and voice synthesis.

## 📂 Structure

```
voice-ai-assistant/
├── 📁 voice-assistant/        # 🎙️ RAG-powered voice assistant
│   ├── main.py                # Main assistant logic
│   ├── rag_system.py          # RAG retrieval system
│   └── requirements.txt
├── 📁 modular-tts/            # 🔊 Custom TTS training pipeline
│   ├── dataset.py             # Dataset preparation
│   ├── train.py               # Model training
│   ├── inference.py           # Inference pipeline
│   └── config.json            # Training configuration
└── 📁 docs/
```

## 🚀 Quick Start

```bash
git clone https://github.com/Emrecaglar05/voice-ai-assistant.git
cd voice-ai-assistant/voice-assistant
pip install -r requirements.txt
python main.py
```
